--------------------------------------------------------------------------------

Bibliotecas utilizadas da linguagem Python
1 - math
2 - matplotlib
3 - random

--------------------------------------------------------------------------------

Pré requisitos:
1 - Python instalado na versão 3.x
2 - Biblioteca matplotlib compatível com a versão do Python
    https://matplotlib.org/users/installing.html

--------------------------------------------------------------------------------

Por ser uma linguagem interpretada o algoritmo escrito em python não necessita
de compilação, deve ser executado via terminal com o seguinte comando:

python caixeiro_genetico.py

No caso de mais de uma versão do python instalada, especificar a versão na
linha de comando

python3 caixeiro_genetico.py
